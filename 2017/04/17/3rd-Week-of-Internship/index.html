3rd Week of Internship

It’s been 2 weeks since I started my internship at MPIWG. Time flies. To be honestly, I didn’t do much things so far. I’m trying to understand the project currently hold by department III, and get familiar with the tools used in the project.

My colleague, Calvin, introduced me “Trello” to track work logs. It’s also easy to implement “SCRUM” on it. There are 5 parts on the dashboard: Backlog, To Do, In Progress, Test and Done. Each of part has some regulations to follow. For example, “To Do”, “In Progress” and “Test” have a limit of WIP. Working time of each task listed on it shouldn’t last longer than 1 day, maximum 2 days, otherwise it has to be broken down to small peaces.

MPIWG uses Mercurial instead of git for source control management. I didn’t hear about it before coming here. After google it, I noticed it’s a relative famous source code control tool in reasearch institutes. It’s really similar as git, just replacing the git to hg.

I’m trying to master docker recently, so I can build a mirror website as I mentioned in previous article. I think my progress is kind of late. Let me work on it harder. Stay on tune!
